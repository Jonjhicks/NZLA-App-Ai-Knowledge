# Can I use Liquid Boost for rust on the lawn?

Yes, as part of a combined treatment. Rust often shows up on nitrogen-starved lawns through late summer and autumn. The fungicide knocks the rust back, the nitrogen push helps the lawn grow past the damage.

Standard rust response programme:
1. Fungus Pro foliar spray to control the active rust infection.
2. Liquid Boost at the same time or within a few days for the nitrogen push.
3. Tank-mix is fine, the two products are compatible.
4. Mow the affected area in the following week (catching the clippings if rust is heavy, to remove the spore load).
5. Reassess after 2 weeks. If rust is gone and lawn is greening up, you're done. If still some rust, second Fungus Pro application.

The reason the combination works: rust thrives on weak slow-growing turf. Fungus Pro alone removes the rust but if the lawn stays underfed, more rust shows up shortly after. Liquid Boost pushes growth so the lawn grows out of the susceptible state.

Going forward, prevent recurrence by keeping nitrogen levels adequate through late summer and autumn. A granular application of All Seasons in March covers most of the year's susceptibility window. Foliar Liquid N or Charger between granular applications keeps the lawn pushing.
